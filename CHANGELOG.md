# Changelog

All notable public changes to ScrapeStudio are documented here. The project follows semantic versioning for published releases.

## [Unreleased]

- Clarified live-release and security-reporting status across public documentation.
- Added a three-step playground path, audience-based documentation routes, Persian repository overview, support routes, and retained `v1.0.0` release evidence.

## [1.0.0] — 2026-07-14

### Added

- Bilingual English/Persian no-code extraction workspace with first-class RTL/LTR and mobile UX.
- Secure Cloudflare Worker fetch gateway with SSRF controls, redirect validation, quotas, timeouts, HTML size limits, and privacy-safe logging.
- Detached client-side extraction for tables, links, images, headings, metadata, and custom multi-field selectors.
- Local IndexedDB recipes and history, safe CSV/JSON export, and template-based Python/JavaScript code generation.
- Bounded Web Worker detection of repeated page structures.
- Original built-in product, table, and article playgrounds requiring no external request.
- Automated unit, integration, E2E, accessibility, security, dependency, build, and bundle-budget gates.
- Manual, protected Cloudflare Pages and Workers production deployment pipeline.
